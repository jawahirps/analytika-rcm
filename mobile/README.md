# Analytika Mobile

Expo React Native UI for the Analytika product surfaces.

## Run

```bash
cd mobile
npm install
npm run ios
```

Use `npm run android` for Android or `npm run web` for a browser preview.

## Current Screens

- Login
- Facility status dashboard
- RCM dashboard with collapsible sections

The app currently uses local sample data. Wire the API calls to the ASP.NET Core backend after the native UI is approved.
